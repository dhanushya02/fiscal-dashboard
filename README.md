# fiscal-dashboard
Interactive Fiscal Deficit Dashboard A stress-testing tool built with Streamlit and Docker to visualize the impact of revenue shortfalls on national fiscal targets for 2025-2027. Includes baseline comparisons and deficit-to-GDP projections.
# Fiscal Dashboard Pro
This dashboard allows users to simulate economic scenarios.

### How to use:
1. Upload the `Master_Fiscal_Dataset.csv`.
2. Use the sidebar slider to set a "Revenue Shortfall %".
3. View the updated 2026-27 projections and the impact on the Fiscal Deficit.
